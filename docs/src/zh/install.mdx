# 安装

## NPM

推荐使用 NPM 安装组件。

```shell
npm install react-ellipsis-component
# or yarn
yarn add react-ellipsis-component
```

## CDN

```html
<script src="https://unpkg.com/react-ellipsis-component/dist/react-ellipsis-component.cjs.production.min.js"></script>
```
