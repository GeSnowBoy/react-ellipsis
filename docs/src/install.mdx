# Install

## NPM

Installing with npm is recommended.

```shell
npm install react-ellipsis-component
# or yarn
yarn add react-ellipsis-component
```

## CDN

```html
<script src="https://unpkg.com/react@16/umd/react.development.js"></script>
<script src="https://unpkg.com/react@16/umd/react-dom.development.js"></script>
<script src="https://unpkg.com/react-ellipsis-component/dist/react-ellipsis-component.cjs.production.min.js"></script>
```
